## [JavaScript 原生学习笔记](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
> **介绍**：JavaScript 是一种广泛用于网页开发的脚本语言，能够实现动态内容和用户交互。它可以在浏览器中运行，控制网页元素、
> 处理事件、与服务器通信，并构建丰富的用户界面，成为现代前端开发的核心技术之一。
